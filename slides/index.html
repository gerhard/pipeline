## A pipeline to get you excited https://pipeline.gerhard.io Note: I believe that everything that can be automated, should. I believe that shipping code should happen with minimal effort, none should be doing it manually. I know that the most successful teams are the ones which are empowered to respond immediately as they learn more about the problems which they are solving. Such teams cannot respond fast enough without an adequate automation story. The humble pipeline is the heart of this automation.

## [Imagine if everything was](#/3) ## automated by [a pipeline](#/3) Note: Imagine if all the boring and repetitive work of getting your code to your users was automated. Some of you might already be doing this, and some might be doing it for many years, maybe even a decade. Now imagine if every single trivial task was automated by some pipeline. I'm thinking testing, compiling, packaging, distributing, benchmarking, scaling and everything else in between. Imagine if anything that can be run in a shell environment was automated... Imagine all the things that you could be doing with the time that this would gain you...
